# wit-wiki：智讯知识库 + AI 顾问

智讯策划咨询的品牌营销知识库，支持 Claude Code / OpenClaw / WorkBuddy 等 AI Agent 调用。

## 快速安装

```bash
git clone git@gitee.com:csmyzi/wit-wiki.git
cd wit-wiki
./install.sh
```

脚本自动检测本机安装的 AI Agent 平台，将 Skill 和知识库部署到对应目录。

## 使用

在 AI Agent 中提问品牌/营销/案例相关问题即可，例如：

- 「智讯做过哪些城市品牌案例？」
- 「反碎片方法论的核心逻辑是什么？」
- 「万科的项目有什么共性？」

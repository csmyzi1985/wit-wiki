# 碧桂园东北区域2017春季品牌发布会整合营销案例

## 背景

2017年是碧桂园扎根东北的第十一年，也是其第二个十年的开端。公司希望改变以往“老百姓买得起的好房子”的单一形象，向市场展示“诗人”、“专家”、“幸福”碧桂园的全新品牌定位。作为数字代理商，需要在去中心化的移动互联时代，帮助碧桂园在沈阳市场进行一次品牌焕新的重磅发声。[R: _智讯案例_复盘碧桂园品牌发布会整合营销攻_心_计]

## 洞察

在信息碎片化的“秒时代”，传统的广告和资讯难以深度触达用户。人性的触点是直达心头最短的路，需要通过创造与城市、与人相关的情感共鸣点，才能有效抓住注意力并传递品牌理念。[R: _智讯案例_复盘碧桂园品牌发布会整合营销攻_心_计]

## 策略

以“致城市·心生长”为核心主题，采用“攻心式”整合营销策略。通过创造一个贯穿始终的[[超级符号理论]]——“镜子”，从引发城市思考、制造悬念、到情感共鸣，层层递进，最终落脚于品牌发布会，完成从话题预热到品牌理念全面展示的闭环。[R: _智讯案例_复盘碧桂园品牌发布会整合营销攻_心_计]

## 执行

1.  **城市话题引导（镜子事件）**：在沈阳太原街、铁西商圈等公共场所设置近百面镜子，引发“城市与人的成长”话题讨论，获得数十家媒体报道。[R: _智讯案例_复盘碧桂园品牌发布会整合营销攻_心_计]
2.  **城市悬念公关（神秘盒子）**：延续“镜子”符号，打造巨型神秘镜面盒子，空降沈阳多个繁华地段，引发全民猜想与媒体跟踪报道。[R: _智讯案例_复盘碧桂园品牌发布会整合营销攻_心_计]
3.  **[[社交媒体营销策略]]走心营销（H5）**：推出以“致敬东北人”为主题的H5，采用一镜到底的创意方式，讲述城市与人共同成长的故事，寻求情感共振。[R: _智讯案例_复盘碧桂园品牌发布会整合营销攻_心_计]
4.  **联动传播（预热视频）**：联合行业媒体及碧桂园内部高管，共同诠释品牌理念，并推出“致城市·心生长”品牌宣传片预热视频。[R: _智讯案例_复盘碧桂园品牌发布会整合营销攻_心_计]
5.  **悬念倒计时**：发布会前，线上围绕“？”符号进行悬念倒计时宣传。[R: _智讯案例_复盘碧桂园品牌发布会整合营销攻_心_计]
6.  **品牌发布会落地**：3月25日，在盛京文化艺术中心举办发布会。现场设置沈阳首个室内智慧BOX体验区、360°环幕TED式演讲、3D全息投影开场秀等爆点，并发布碧桂园·御长白及碧桂园·公园上城两个新项目案名。[R: _智讯案例_复盘碧桂园品牌发布会整合营销攻_心_计]

## 效果

本次整合营销活动成功实现了品牌焕新。前期话题与悬念事件引发了广泛的社会关注和媒体报道。发布会当天，吸引了沈城36家媒体及近20家微信大V主动参与报道，有效提升了碧桂园在东北区域的品牌形象，从“性价比”标签向更具情感与人文关怀的“诗人”、“专家”、“幸福”形象成功转型。[R: _智讯案例_复盘碧桂园品牌发布会整合营销攻_心_计]

# 产品系叙事 · 概念库（深化版）

> ⚪ 竞品分析 · 仅供内部参考
>
> **核心洞察：** 产品系叙事的关键不是"不同城市讲同一个故事"，而是"同一个灵魂在每座城市找到不同的肉身"。

---

## 方法论总纲

### 产品系叙事的通用公式

```
命名 = [核心符号] + [城市文化锚点] + [后缀]
叙事 = DNA（刚性不变） × 在地化（柔性替换）
进化 = 原点定义 → 区域复制 → 跨省升级
```

### 三个产品系的横向对比

| | 凤起系 | 海棠系 | 云庐系 |
|------|--------|--------|--------|
| 核心DNA | 凤凰起飞，城市焕新 | 海棠绽放，美好降临 | 庐隐于市，中国人的精神归宿 |
| 命名规则 | 凤起 + [锚点] + 鸣 | [锚点] + 海棠 | [锚点] + 雲庐 |
| 覆盖城市 | 8城 | 杭州→西安 | 9城全国 |
| 叙事进化 | 省内扩散→跨省→产品迭代 | 一城多花→跨省合奏 | 全国快速复制 |

## 凤起系：凤凰的八次起飞

### 命名系统

```
凤起 + [城市文化锚点] + 鸣
  ↑         ↑            ↑
 恒定      每城不同      恒定
```

| 城市 | 锚点 | 名字 | 文化坐标 |
|------|------|------|----------|
| 杭州 | 潮 | 凤起潮鸣 | 老十门内、护城河畔 |
| 宁波 | 潮 | 凤起潮鸣 | 老城核心 |
| 台州 | 潮 | 凤起潮鸣 | 山海、和合文化 |
| 义乌 | 潮 | 凤起潮鸣 | 迎恩门、渌波亭 |
| 天津 | 悦 | 凤起悦鸣 | 鼓楼老城厢（1404年） |
| 长沙 | 麓 | 凤起麓鸣 | 岳麓书院、湘江 |
| 苏州 | 潮 | 凤起潮鸣 | 金鸡湖（湖居版迭代） |
| 慈溪 | 潮 | 凤起潮鸣 | 潮塘江 |

### 命名洞察

"鸣"是凤起系的核心后缀——凤凰起飞必有声响。但"鸣"前面的字决定了城市故事的角度：
- **潮鸣**（最多）：城市与时代的共振——最通用的城市叙事
- **悦鸣**（天津）：需要不同的声调——天津是"有凤来仪"的喜悦
- **麓鸣**（长沙）：需要不同的空间——长沙是山脚下的凤凰（岳麓山）
- **湖居版凤起**（苏州）：第一次不是"市中心"而是"湖边"——产品形态迭代

### 叙事进化规律

```
杭州（原点）
  ↓ 确立DNA：凤起 = 城市核心 + 高端序列
宁波/台州/义乌（省内扩散）  
  ↓ 验证模式：每城找不同锚点，但"凤起潮鸣"四字不变
天津/长沙（跨省）
  ↓ 开始微调：天津用"悦"不是"潮"，长沙用"麓"不是"潮"
苏州（产品迭代）
  ↓ 形态突破：第一个湖居版凤起——"一面都市三面湖"
```

**💡 启发：** 产品系不需要每城都叫同一个名字。核心符号（凤起）和核心后缀（鸣）保持不变，中间的地域锚点可以随城市变化。当锚点改变时，意味着产品形态或城市关系也发生了变化。

## 海棠系：一朵花的六次绽放

### 命名系统

```
[城市文化锚点] + 海棠
      ↑              ↑
   每城不同         恒定
```

| 项目 | 锚点含义 | 叙事角度 |
|------|----------|----------|
| 熙海棠 | 熙 = 光明、兴盛 | 云谷新生板块 → "以一个项目定义一个板块" |
| 锦海棠 | 锦 = 织锦 | 杭州织锦文化 + 玉湖 → "绣一湖花开" |
| 玉海棠 | 玉 = 良渚玉琮 | 良渚五千年文明 → "良渚之心，流光溢彩" |
| 海棠三篇 | — | 北部新城三块地 → "理想向上，好运杭州" |
| 海棠三章 | — | 西安三块地：月映/春熙/凤鸣 → "化龙西安" |

### 命名洞察

海棠系的智慧在于：**每一朵海棠的名字就是它的故事核心。**
- 熙海棠：板块是"新生"的，所以海棠是"熙"（光明的开始）
- 锦海棠：板块有玉湖和织锦文化，所以海棠是"锦"（编织繁华）
- 玉海棠：板块有良渚玉文化，所以海棠是"玉"（文明的光彩）

**每个命名 = 板块DNA × 海棠意象**

### 叙事进化规律

```
杭州：一朵花 → 六朵花（同城多板块）
  ↓ 每朵花在不同的"春天"开放
  ↓ 单城验证：同一产品系可以在同一城市的不同板块各有叙事
西安：三章（跨省合奏）
  ↓ 从"一朵花"升级为"三章"——叙事规模升级
  ↓ 三个地块不再独立命名，而是组成"海棠三章"
```

**💡 启发：** 当产品系在同一城市落地多个项目时，每个项目需要独立的叙事锚点（不同的"春天"）。当产品系跨省时，可以升级叙事规模——从"一朵花"到"三章合奏"。

## 云庐系：隐逸的九种姿态

### 命名系统

```
[城市文化锚点] + 雲庐
      ↑              ↑
   每城不同         恒定
```

| 城市 | 名字 | 隐逸哲学 | 叙事角度 |
|------|------|----------|----------|
| 杭州 | 咏溪雲庐 | 赞咏 | "杭州的赞咏"——对城市的赞美 |
| 合肥 | 咏溪雲庐 | 赞咏 | "庐州的赞咏"——同名但城市不同 |
| 扬州 | 凤鸣雲庐 | 贵隐 | "一园双庐，贵隐南河下" |
| 嘉兴 | 湖境云庐 | 上善若水 | "上善若水，隐逸嘉兴" |
| 北京 | 西山云庐 | 贵 | "贵以西山，水流云在" |
| 天津 | 水西雲庐 | 隐逸 | "天津的隐逸" |
| 武汉 | 湖畔雲庐 | 风雅 | "重拾武汉的风雅" |

### 命名洞察

云庐系最特别：**核心DNA是"隐逸"，但每座城市的隐逸方式完全不同。**
- 杭州是"赞咏"（诗意的隐逸）——因为杭州本身就是诗
- 扬州是"贵隐"（有身份的隐逸）——因为扬州的运河文化是"贵重"的
- 嘉兴是"上善若水"（水的智慧）——因为嘉兴是水乡

**每个命名 = 这座城市怎么理解"隐逸"**

### 💡 产品系叙事的通用启示

1. **两段式命名法**：核心符号（凤起/海棠/雲庐）+ 城市文化锚点。前者保证辨识度，后者保证在地性。
2. **DNA刚性，在地化柔性**：凤起永远是焕新，海棠永远是绽放，云庐永远是隐逸。但每城焕新的方式、绽放的姿态、隐逸的哲学可以完全不同。
3. **进化三步走**：原点定义DNA → 区域内密集复制验证 → 跨省扩张并可能迭代产品形态。
4. **命名即叙事**：海棠系最典型——每个命名（熙/锦/玉）直接定义了这个项目的核心故事角度。
